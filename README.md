# External Brain in Markdown (English)

This is the data repository that stores Markdown exports from the Scrapbox project [nishio-en](https://scrapbox.io/nishio-en/).

The Markdown files in this repository are generated by GitHub Actions. The export and conversion code is maintained separately in [nishio/from_scrapbox](https://github.com/nishio/from_scrapbox).

## Contents

- `pages/`: one Markdown file per Scrapbox page
- `.github/workflows/update_markdown.yml`: the workflow that refreshes `pages/`
- `README.md`: this repository guide

## Updates

The `Update English Markdown Files` workflow normally runs every day at JST 05:30.

The workflow:

1. checks out this data repository
2. checks out [nishio/from_scrapbox](https://github.com/nishio/from_scrapbox) into `_tools/from_scrapbox`
3. exports the `/nishio-en` Scrapbox project
4. converts the exported JSON to Markdown
5. syncs the generated files into `pages/` and commits any changes

To run an update manually, open GitHub Actions and run `Update English Markdown Files`.

## Notes

- `pages/` is generated output. Manual edits may be overwritten or deleted by the next update.
- The Scrapbox export requires the repository secret `SID`.
- Change the export or conversion logic in [nishio/from_scrapbox](https://github.com/nishio/from_scrapbox), not in this data repository.
