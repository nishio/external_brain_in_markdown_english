title: "Simple rules for newcomers to PastaLogic"
---

command preparation phase
- Decide on the first or second move
- Make life to 3 points.
- Line up command cards.
    - Bug
    - AddFlag
    - Subroutine
    - MoveFlag
    - Increment
- Place the cursor on the first card

flag placement phase
- The first player chooses a card on which his/her flag has not yet been placed and places his/her flag on it.
- The latter hand does likewise.
- Repeat three times.

- Flags are arranged in columns in the order they are placed on the command card.
    - This means that "the flag set first has a higher priority.
